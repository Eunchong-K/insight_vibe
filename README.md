# 단어 학습 플래시 카드 앱

## 📚 프로젝트 소개

**✨ 완전히 새롭게 디자인된 단어 학습 플래시 카드 앱!**

PC와 모바일에서 모두 사용 가능한 반응형 플래시 카드 애플리케이션입니다. 4지선다형 퀴즈를 통한 효과적인 단어 학습을 지원하며, 현대적인 디자인과 부드러운 애니메이션을 제공합니다.

## 🎯 주요 기능

### ✨ 핵심 기능
- **🔄 플래시 카드 방식**: 단어를 보여주고 **의미는 처음부터 바로 표시**
- **📝 4지선다형 퀴즈**: **다른 단어의 의미**와 섞어서 선택지 제공
- **⏰ 자동 진행**: 정답 선택 후 **3초 후 자동**으로 다음 단어로 진행
- **🎲 랜덤 단어 선택**: 매 학습마다 랜덤으로 10개 단어 선택
- **💾 점수 기록 및 저장**: localStorage를 활용한 학습 기록 저장
- **📱 모바일 최적화**: **세로 스크롤 없는** 완전한 화면 구성

### 🎨 디자인 특징
- **✨ 현대적인 그라데이션**: 아름다운 배경 그라데이션
- **🎭 유리 효과**: backdrop-filter를 활용한 유리 효과
- **💫 부드러운 애니메이션**: 카드 플립 및 화면 전환 애니메이션
- **🎯 직관적 UI**: 사용하기 쉬운 인터페이스

### 📊 학습 관리
- **📈 학습 기록 보기**: 이전 학습 점수와 날짜 확인
- **🔄 재학습 기능**: 같은 세션 다시 보기 가능
- **⚡ 즉시 피드백**: 정답/오답 즉시 시각적 표시
- **🎉 결과 화면**: 학습 완료 후 성취감 제공

## 🚀 사용 방법

### 1. 시작하기
- 웹 브라우저에서 `index.html` 파일 열기
- "학습 시작" 버튼 클릭

### 2. 학습 진행
- **단어와 의미 확인**: 카드에 표시된 단어와 의미를 바로 확인
- **정답 선택**: 4개 선택지 중 정답 선택 (다른 단어의 의미가 포함됨)
- **자동 진행**: 정답 선택 후 3초 후 자동으로 다음 단어로 진행
- **수동 진행**: "다음 단어" 버튼으로 즉시 진행도 가능

### 3. 결과 확인
- 10개 단어 학습 완료 후 점수 확인
- 학습 기록이 자동으로 저장됨

### 4. 기록 보기
- 상단의 "학습 기록" 버튼 클릭
- 이전 학습 점수와 날짜 확인 가능

```

├── data/
│   └── words.json         # 단어 데이터 (20개 단어 포함)

```

## 📖 단어 데이터

`data/words.json` 파일에 단어, 품사, 의미, 선택지가 포함되어 있습니다.

### 단어 데이터 형식
```json
{
    "word": "abundant",
    "type": "형용사",
    "meaning": "풍부한, 많은",
    "options": ["부족한", "적은", "풍부한", "드문"]
}
```

### 포함된 단어
- 20개의 고급 영어 단어
- 각 단어별 4개의 선택지 (다른 단어의 의미 포함)
- 정답과 오답 명확히 구분

## 🐛 문제 해결

### 일반적인 문제
1. **단어가 표시되지 않음**: `data/words.json` 파일 경로 확인
2. **점수가 저장되지 않음**: 브라우저의 localStorage 지원 확인
3. **자동 진행이 작동하지 않음**: JavaScript 타이머 확인

### 디버깅 도구
- **브라우저 개발자 도구**: F12
- **콘솔 로그**: 자세한 실행 정보
- **네트워크 탭**: 데이터 로딩 상태 확인
